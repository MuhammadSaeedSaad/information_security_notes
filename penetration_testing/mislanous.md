# create virtual environmet (python3 venv

## 1. create file
```bash
mkdir python_evn
```

## 2. install dependency
```bash
sudo apt install python3.10-venv
```

## 3. create the venv
```bash
python3 -m venv <env_dir_name>
```

## 4. activate the venv
```bash
source ./bin/activate
```

## 5. install requirements
```bash
pip install -r requirements.txt
```
